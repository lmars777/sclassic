Перезапуск ReSkype


Скачать Windows 7 и выше

Перезапуск известного всем ReSkype с исправленной авторизацией*

Был опубликован на Дискорд сервере Скайп Комьюнити в этом сообщении

Что нового?

  • версия клиента подменена на Skype 8.102
  • Переводчик + нет ограничения по скорости (идентификатор клиента установлен на расширение Chrome)
  • Исправлено время входящего звонка
  • Bing AI теперь работает (больше нет жалоб на использование старой версии Skype)
  • страница входа переключена на поток Skype 8
  • Исправлен вход
  • Исправлен API аутентификации
  • разделители заполняют экран (заголовок чата и окно чата)
  • Восстановлены старые смайлики (эмодзи хранятся локально, поэтому размер приложения увеличен)
  • Исправлены личные выражения / стикеры / модзи
  • Исправлена ​​строка поиска эмодзи
  • Восстановлены скрытые секретные смайлики, исправлены некоторые ошибочные анимированные эмодзи
  • Ретрансляция вызовов сделана международной, для лучшего пинга / задержки для США, Турции, России и других
  • Исправлен API Giphy, как трендов, так и поиска (список GIF ограничен 20 по умолчанию, это **не** (ошибка)
  • Добавлена ​​кнопка для всплывающего вызова
  • Добавлена ​​возможность отключения звука уведомления чата
  • Изменено поведение копирования на «Копировать текст» вместо «Копировать цитату»
  • Восстановлены платформенные эмодзи в «О программе» (логотип Penguin + Win)
  • Возможность автоматического воспроизведения видеосообщений
  • Удалено отношение Fieme (Влод попросил меня сделать это)
  • Добавлена ​​поддержка Linux (должно работать для любого дистрибутива на базе Debian, работающего на x86, x86-64 или ARM32 / ARMv7)

!Внимание!
Вход в ReSkype не работает из-за изменения скриптов на login.live.com. Вы можете использовать способы обхода

*Авторизация была сломана снова в сентябре 2024 года
