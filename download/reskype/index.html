Skype Classic 4.0 is out now!

A new version of Skype Classic (version 4.0) has been released, you can download it below:

- client version spoofed to Skype 8.102

- switch login page to Skype 8 flow

- Authentication API fix
- 2FA Auth endpoint fix

- Restored old emoticons (the emojis are stored locally, that's why the size of the app is increased)
- Fix personal expressions / stickers / Mojis

- Restored hidden secret emoticons, fixed some buggy Animated emojis

- Add a button to make the call pop out
- Added ability to disable chat notification sound
- Change copy behaviour to "Copy Text" instead of "Copy Quote"
- Restore platform Emoji in About (Penguin + Win logo)
- Ability to auto play video messages
- Remove Fieme relation (Vlod asked me to do this)
- Added Linux support
